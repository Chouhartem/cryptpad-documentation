Collaboration / Social¶

Profil¶

Utilisateurs enregistrés

Chaque utilisateur enregistré sur CryptPad a une page de profil. Celle-ci est accessible dans le menu utilisateur :

  • Menu utilisateur (avatar en haut à droite) > Profil.

Profil personnel¶

Sur la page de profil personnel, les éléments suivants sont personalisables :

  • L'avatar avec Importer un nouvel avatar.

  • La description personnelle avec Ajouter une description.

  • L'adresse web avec Ajouter un lien vers votre site web.

Partage :

  • Voir mon profil : Aperçu de la page de profil comme elle apparaît aux autres utilisateurs.

  • Partager : Copier le lien vers votre profil.

Profil utilisateur¶

Pour voir le profil d'un autre utilisateur :

  • Liste d'utilisateurs d'un document où ils sont connectés > Clic sur leur nom.

  • Si l'utilisateur est dans votre liste de [contacts]. Menu utilisateur (avatar en haut à droite) > Contacts > Double-Clic sur leur nom dans la liste.

Sur la page de profil d'un autre utilisateur :

  • Envoyer une demande de contact : Ajoute cet utilisateur à vos :ref:`contacts <contacts>`une fois votre demande acceptée.

  • Masquer : Bloque les notifications et les messages venant de ce compte. L'utilisateur ne sera pas informé que vous l'avez masqué.

  • Copier la clé publique : Copie la clé utilisée par les administrateurs d'instance et/ou sur les instances avec abonnement.

Contacts¶

Utilisateurs enregistrés

Les contacts CryptPad simplifient et sécurisent la collaboration.

Ajouter un contact¶

Pour voir le profil d'un autre utilisateur :

  • Envoyer une demande de contact.

En partageant votre profil :

  1. Copier le lien : menu utilisateur (avatar en haut à droite) > Profil > Partager.

  2. Coller et envoyer par le moyen de votre choix (préférez un outil de communication sécurisé).

Gérer les contacts¶

Pour accéder à la page Contacts :

  • menu utilisateur (avatar en haut à droite) > Contacts.

La liste de contact est à gauche de la fenêtre. Pour chaque contact :

  • : Masquer les messages et notifications.

  • : Supprimer.

  • : Indique que le contact est actuellement connecté.

Chat avec les contacts¶

Sur la page Contacts, Clic sur un contact dans la liste ouvre la conversation avec cette personne dans la fenêtre principale.

Écrire des messages dans le champ en bas de la fenetre et les envoyer avec ou Entrée.

Récupérer l'historique du chat avec ou l'effacer avec .

Équipes¶

Utilisateurs enregistrés

Les équipes CryptPad sont des espaces partagés entre plusieurs utilisateurs. Une équipe possède son propre stockage CryptDrive, un chat et une liste de membres dont l'accès peut être régulé.

Note

Le nombre d'équipes dont un utilisateur peut être membre est limité à 3 pour des raisons de performance.

Pour créer une équipe :

  1. Menu utilisateur (avatar en haut à droite) > Équipes.

  2. Nouvelle équipe.

Pour ouvrir une équipe existante :

  1. Menu utilisateur (avatar en haut à droite) > Équipes.

  2. Équipes > Bouton Ouvrir de l'équipe souhaitée.

Drive d'équipe¶

L'espace de stockage de l'équipe est similaire au CryptDrive utilisateur, mais partagé entre les membres.

Membres¶

L'onglet Membres permet de gérer les membres d'une équipe.

Inviter des membres¶

Pour inviter des membres dans une équipe :

  • Onglet Membres > Inviter des membres.

Onglet Contacts : Sélectionner des contacts CryptPad pour les inviter. Les destinataires reçoivent une notification et peuvent accepter l'invitation.

Onglet Lien : Copier un lien à envoyer (de préférence par un moyen sécurisé). Le lien créé est à usage unique, il est invalidé une fois qu'un utilisateur l'utilise pour rejoindre l'équipe.

  • Nom temporaire : Utilisé pour reconnaître l'invitation dans la liste des invitations en attente.

  • Mot de passe : Protection du lien contre une éventuelle interception. (facultatif)

  • Message personnalisé* : message visible par le destinataire avant de décider d'accepter l'invitation.

Permissions :

Lecture-seule : accéder aux dossiers et documents sans les modifier.

Édition : créer, modifier et supprimer des dossiers et documents.

Gérer les membres : inviter et révoquer les membres, changer les rôles jusqu'à Administrateur.

Gérer l'équipe : changer le nom et l'avatar de l'équipe, ajouter ou révoquer les propriétaires de l'équipe, supprimer l'équipe.

Administration¶

La liste des membres permet de gérer le rôle de chacun. Les propriétaires et administrateurs de l'équipe peuvent gérer les membres de rôle égal ou inférieur. Pour chaque membre :

: Promouvoir au rôle supérieur.
: Rétrograder au rôle inférieur.
: Expulser de l'équipe.

Chat¶

Le chat de l'équipe est similaire au chat avec un contacts, mais partagé entre tous les membres.

Onglet Administration¶

Propriétaires de l'équipe

  • Clé publique de signature : Utilisée pour identifier l'équipe sur les instances avec abonnements.

  • Nom de l'équipe : Changer le nom de l'équipe.

  • Avatar de l'équipe : Importer/modifier un avatar pour l'équipe.

  • Suppression de l'équipe : Supprime définitivement l'équipe et ses documents.
