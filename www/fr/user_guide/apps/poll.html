Sondage¶

L'application Sondage est conçue pour faciliter les décisions dans un petit groupe de personnes de confiance, par exemple pour choisir une date/heure de réunion.

Elle n'est pas adaptée pour des sondages partagés avec un grand nombre de personnes et/ou qui nécessitent des garanties sur la sécurité du vote. En effet, pour voter chaque participant doit avoir les droits d'édition sur le document. Cela permet de modifier ou de supprimer les réponses des autres participants.

Une refonte complète de cette application est prévue pour fin 2020.

Barre d'outils¶

Modifier : Passer en mode édition pour renseigner ou modifier la description du sondage et les options de réponse.

Publier : Sauvegarder les changements à la description et aux options de réponse du sondage.

En mode édition :

Outils : Affiche ou cache la barre d'outils de l'éditeur de texte.

Insérer : Intègre une image dans votre document. Vous pouvez choisir une image dans votre CryptDrive ou en importer une depuis votre ordinateur. Utilisateurs enregistrés

Édition¶

La description du sondage est écrite en syntaxe Markdown. Le bouton Insérer permet d'y ajouter des images.

Les options de réponse sont ajoutées à gauche du tableau. Pour créer une nouvelle option :

  1. Ajouter son titre dans le champ "Option".

  2. Clic sur ou Entrée.

Pour modifier une option, Clic sur le ✐ à droite du titre. Pour la supprimer, Clic sur à gauche du titre.

Vote¶

Chaque vote est comptabilisé dans une colonne du tableau de résultats. Pour ajouter un vote :

  1. Écrire votre nom en titre de la colonne.

  2. Clic sur les cellules des différents choix pour indiquer :

  • ✔ Oui

  • ~ Peut-être

  • ✖ Non

  1. Clic sur Ajouter en haut de la colonne pour la sauvegarder et la comptabiliser dans le total.

Le bouton permet de déverrouiller une colonne et de modifier le nom et les choix. Le bouton vérouille la colonne.

Pour ajouter un commentaire sous le sondage :

  1. Écrire votre texte dans le champ prévu à cet effet "Votre commentaire".

  2. Envoyer
