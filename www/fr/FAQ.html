Foire Aux Questions¶

Comment importer/exporter mes documents vers/depuis un autre logiciel ?¶

Rendre CryptPad compatible avec une large gamme de formats de documents est un enjeu de taille pour l'équipe de développement. Malheureusement, la technologie qui permet de convertir les documents dans le navigateur a encore des limites, surtout en ce qui concerne le respect de la vie privée. Pendant que le travail dans ce domaine se poursuit, les méthodes ci-dessous permettent d'importer et d'exporter depuis/vers d'autres logiciels largement utilisés.

Note

Un simple copier-coller du contenu depuis un autre logiciel vers CryptPad est une première étape facile à essayer pour l'import/export. Veuillez noter que les méthodes ci-dessous peuvent cependant améliorer considérablement les résultats.

Il est possible d'importer des fichiers sur CryptPad pour un stockage et un partage sécurisés, mais cela ne fait pas de chaque fichier un document modifiable par les applications CryptPad.

Les fichiers texte importés peuvent être ouverts en tant que document Code/Markdown depuis le Drive avec Clic droit > Ouvrir dans l'application Code.

Pour importer tout autre type de fichier dans une forme modifiable, veuillez utiliser la fonctionnalité Fichier > Importer dans l'application CryptPad appropriée, comme expliqué ci-dessous.

Import/export de documents texte¶

Pour importer des documents texte (Microsoft Word, Google Docs, LibreOffice Writer, etc.), le format compatible est HTML :

  1. Exporter le document au format HTML depuis un autre logiciel.

  2. Enregistrez le fichier sur votre ordinateur.

  3. Ouvrir un nouveau document texte sur CryptPad (ou un document existant pour remplacer le contenu).

  4. Fichier > Importer et sélectionnez votre fichier.

Pour exporter, utiliser Fichier > Exporter et sélectionner .doc comme format.

Import/export de feuilles de calcul¶

Pour importer des feuilles de calcul, les formats compatibles sont Microsoft Excel .xlsx et OpenDocument .ods. Ces deux formats peuvent être exportés à partir de logiciels courants tels que Google Sheets et LibreOffice Calc :

  1. Exporter le document au format .xlsx ou .ods depuis un autre logiciel.

  2. Enregistrez le fichier sur votre ordinateur.

  3. Ouvrir une nouvelle feuille de calcul sur CryptPad (ou un document existant pour remplacer le contenu).

  4. Fichier > Importer et sélectionnez votre fichier.

Pour exporter, utiliser Fichier > Exporter et sélectionner un format :

  • Le format .csv est le plus largement compatible (à noter qu'il ne contient que les données et non le formatage).

  • Pour utilisation dans Microsoft Excel ou Google Sheets, sélectionner .xlsx.

  • Pour utilisation dans LibreOffice Calc, sélectionner .ods.

  • Pour les cas où aucune autre modification des données n'est nécessaire après l'exportation (présentations, rapports, etc.), sélectionner .pdf.

  • Pour faire une sauvegarde ou pour partager avec un utilisateur sur une autre instance CryptPad, sélectionner .bin.

Je ne peux pas faire X avec un document que j'ai créé¶

Certaines fonctionnalités sur CryptPad sont réservées aux propriétaires du document. Cela inclut :

  • Activer une liste d'accès.

  • Ajouter un mot de passe.

  • Ajouter ou supprimer d'autres propriétaires.

  • Détruire le document.

La propriété d'un document est établie lors de sa création et ne peut pas être modifiée par la suite si le document est créé sans propriétaire.

L'équipe de développement envisage de supprimer cette distinction à l'avenir. Pour le moment, l'option la plus sûre est d'être "propriétaire" de tous les documents que vous créez afin de conserver le contrôle sur ceux-ci, y compris la possibilité de les détruire.

Puis-je utiliser CryptPad sur mobile ?¶

CryptPad est conçu pour fonctionner le mieux possible sur les petits écrans. En fonction des performances de votre appareil, il devrait être possible d'utiliser CryptPad sur un téléphone portable. Des travaux visant à améliorer cet usage ont été entrepris en 2020. Si vous remarquez des points à améliorer à cet égard, veuillez contacter Support ou le signaler sur Github.

Avez vous l'intention de faire une application ?¶

Nous ne prévoyons pas d'application mobile pour les raisons suivantes :

  • Cela augmenterait considérablement la quantité de code à développer et à maintenir, créant de fait plusieurs "versions" de CryptPad pour iOS et Android.

  • CryptPad est un logiciel libre et peut être hébergé par quiconque souhaite proposer ce service. Par conséquent, les utilisateurs d'une application mobile devraient spécifier à quelle instance CryptPad ils veulent se connecter, ce qui serait déroutant. Pour compliquer encore les choses, chaque instance peut exécuter une version différente du logiciel, suivant la dernière mise à jour appliquée par les administrateurs.

Pour résoudre ces problèmes, l'équipe de développement travaille à faire du CryptPad une "application Web progressive". Cela signifie que le logiciel peut être utilisé sur un mobile par le biais du navigateur, se comportant comme une application tout en étant le même que celui qui fonctionne sur les navigateurs de bureau. Cela présente l'avantage de transformer chaque instance de CryptPad en un fournisseur d'application web, plutôt que de faire peser sur l'utilisateur la tâche de choisir la bonne instance.

CryptPad peut-il synchroniser des documents avec mon ordinateur ?¶

La façon dont le chiffrement est actuellement utilisé dans le CryptPad ne permet pas la synchronisation avec le système de fichiers local. Cependant, cette fonctionnalité est régulièrement demandée, et l'équipe de développement espère la rendre disponible à l'avenir.

Quelle est la relation entre CryptPad et OnlyOffice ?¶

L'application CryptPad Tableur est une intégration du Tableur OnlyOffice. Cependant, cela ne concerne que le code côté client, le CryptPad n'utilise pas le serveur de documents OnlyOffice. La collaboration chiffrée du CryptPad, utilisée pour les feuilles de calcul et autres applications, est complètement différente du système de chiffrement utilisé dans certaines parties du OnlyOffice en amont. Certains des outils de conversion de format de fichier du CryptPad sont basés sur le code OnlyOffice, mais un travail important a été effectué pour le faire fonctionner dans le navigateur plutôt que sur le serveur, évitant ainsi de devoir révéler le contenu des documents des utilisateurs lors de la conversion.
