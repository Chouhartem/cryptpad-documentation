
## Compte Utilisateur

### Particularités des comptes CryptPad


#### Avertissements

Avant de créer un compte, il y a deux informations à savoir :

- CryptPad est **chiffré**, ce qui veut dire que **nous ne pouvons pas retrouver votre mot de passe**.

- CryptPad utilise la combinaison **nom d'utilsateur/mot de passe** pour vous identifier, c'est donc pour cette raison que **les noms d'utilisateurs ne sont pas uniques**.



#### Différents types de comptes

Il existe trois types d'utilisateurs sur cryptpad avec ses propres avantages:

- **Utilisateur anonyme**
  - Aucune information ne vous est demandé
  - Accès à toutes les applications proposés par CryptPad
  - Collaboration et partage de votre document avec vos destinataires
  - La durée de stockage est limité à 90 jours sans modification


- **Utilisateur enregistré**
  - Seulement un nom d'utilisateur et un mot de passe vous est demandé
  - Accès à un espace de stockage de 1GB (**uniquement sur cryptpad.fr**)
  - Vos Documents sont permanent sur votre CryptDrive
  - Création d'une [équipe] qui sera séparé de vos documents personnels
  - Gestion de vos documents plus complète (ajout d'un mot de passe, d'une durée de vie et de sa gestion en y étant son [propriétaire])
  - Recevoir des notifications sur les actions vos contacts
  - personnalisation poussé de vos documents (ajout de [mots-clés])
  - création d'un dossier et possibilité de partager tout son contenu
  - Transformation d'un de vos documents en un [modèle] pour vos prochaines créations
  - [Ajouter des contacts] et discuter avec eux dans un salon
  - Personnalisation de votre profil (ajout d'une photo, biographie)
  - Mettre ses fichier personnels sur son CryptDrive


- **Utilisateur premium (uniquement sur cryptpad.fr)**
  - Support d'aide plus rapide
  - Espace de stockage plus grand
  - Aider le développement et la pérennité de CryptPad



### Gestion du compte



#### Inscription

Pour vous inscrire, lisez les [avertissements] et rendez-vous sur la page [d'inscription] pour remplir les informations suivantes :

- {**Obligatoire**} Nom d'utilisateur -> Nom utilisé pour vous connecter sur votre compte (ce n'est pas forcement le même qui est affiché pour les autres utilisateurs).

- {**Obligatoire**} Mot de passe -> Mot de passe utilisé pour vous connecter sur votre compte (de préférence complexe, peut être changer dans les paramètres, n'est pas retrouvable par CryptPad).

- {**Optionnels**} Pads anonyme -> Si vous avez créer des documents dans votre session anonyme, vous pouvez les sauvegarder dans votre compte.

- {**Obligatoire**} Conditions d'utilisation -> Condition obligatoire à respecter quand on s'inscrit à notre service.



#### Connexion

Pour vous connecter sur votre compte, rendez vous sur la page de [connexion] et remplissez les informations suivantes :

- {**Obligatoire**} Nom d'utilisateur -> Nom utilisé lors de votre inscription.

- {**Obligatoire**} Mot de passe -> Mot de passe utilisé lors de votre inscription ou que vous avez changé dans vos paramètres.

- {**Optionnels**} Pads anonyme -> Si vous avez créer des documents dans votre session anonyme, vous pouvez les sauvegarder dans votre compte.

#### Paramètres du compte



##### Compte

- {**Connecté**} Nom d'utilisateur -> Nom utilisé lors de votre inscription.

- {**Connecté**} Clé publique de signature -> Clé utilisé pour vous identifier et permettre à vos amis de vous ajouter dans leur contact.

- Nom affiché -> Nom utilisé pour vous reconnaître dans un document et que vous pouvez modifier.

- Langue -> Modifie la langue d'affichage de CryptPad.

- Astuces -> Réinitialise les astuces données dans CryptDrive.

- {**Connecté**} Changer de mot de passe -> Permet de changer son mot de passe pour la prochaine connexion.

- {**Connecté**} Suppression du compte -> Supprime définitivement son compte et ses documents.

##### Confidentialité 

- {**Connecté**} Fermer les autres sessions -> Vous déconnecte de toutes les autres session où vous êtes connecté sauf celui où vous avez activer cette fonctionnalité.

- Stockage des pads dans CryptDrive -> Vous permet de choisir si vous voulez enregistrer automatiquement ou manuellement avec ou sans notification les documents de vos contacts que vous visitez dans votre CryptDrive.

- Liens Sécurisés -> Fonctionnalité qui permet de partager les clés de vos documents sans que votre navigateur, ni même les extensions ne l'enregistrent et ne la connaissent.

- Retour d'expérience -> Permet d'envoyer automatiquement à CryptPad des retours d'expérience afin de les aider à améliorer l'expérience des utilisateurs.

##### CryptDrive

- {**Connecté**} Doublons des pads dont vous êtes propriétaire -> Permet de cacher les sauvegardes des documents qui sont dans un dossier partagé. Ils reviendront quand vous aurez supprimés les documents du dossier partagé.

- Miniatures -> Désactive la création des miniatures de vos documents afin de gagner de la place dans votre navigateur et éviter un ralentissement de celui-ci.

- Sauvegarde -> Permet de sauvegarder les clés de vos documents afin de pouvoir les restaurez dans une autre session pour y accéder en ligne ou alors vous pouvez télécharger tout le contenue de votre drive directement dans votre ordinateur.

- {**Connecté**} Importer -> Si vous avez créer des documents dans votre session anonyme, vous pouvez les sauvegarder dans votre compte.

- Effacer l'Historique -> Vous pouvez supprimer l'historique de votre disque et les notification afin de gagner de l'espace de stockage (n'affecte pas l'historique des documents).

##### Curseur

- Couleur du curseur -> Change la couleur de votre curseur pour vous reconnaître dans un document.

- Partager la position de mon curseur -> Permet aux utilisateurs de voir la position de votre curseur dans un document.

- Afficher les curseurs des autres utilisateurs (BETA) -> Une fonctionnalité en BETA qui vous permet de voir les curseurs des autres utilisateurs s'ils ont acceptés de le partager.

##### Documents texte

- Largeur de l'éditeur de texte -> Réduit la largeur de l'éditeur de texte en mode page.

- Vérification orthographique -> Cette option souligne les fautes d’orthographes et vous propose des corrections.

- Notifications de commentaires -> Désactive les notifications lorsqu'un utilisateur réponds à vos commentaires.

##### Code

- Indentation dans l'éditeur de code (nombre d'espaces) -> Options qui permet de choisir le nombre d'espaces effectués quand on appuie sur la touche tabulation.

- Utiliser des tabulations au lieu d'espaces -> Ne considère plus la touche de tabulation comme une barre d'espace personnalisable.

- Fermer automatiquement les parenthèses -> Ferme automatiquement une parenthèse quand on en ouvre une.

- Taille de la police dans l'éditeur de code (px) -> Permet de régler la taille du texte dans l'éditeur de code.

- Vérification orthographique -> Cette option souligne les fautes d’orthographes et vous propose des corrections.

##### Abonnement

- Vous redirige vers la page d'abonnement.



### Support

- Titre du ticket -> Résumé en une ligne de pourquoi vous nous contacté.

- Taper votre message... -> Description complète de pourquoi vous nous contacté.

- Fermer le ticket -> Considère la discussion comme résolue (mais ne la supprime pas).

- Supprimer le ticket -> Supprime définitivement la discussion.



### Notification

- {**Connecté**} (x) -> Supprime la notification.

- {**Connecté**} Poubelle -> Supprime l'ensemble des notifications.



### Abonnement

- {**Connecté**} (x) -> Supprime l'abonnement actuel définitivement.

- {**Connecté**} Montrer uniquement les abonnements actifs -> Cache vos abonnement qui ne sont plus à jours.

- {**Connecté**} {**Obligatoire**} Clé du bénéficiaire -> Clé publique de votre contact à mettre afin qu'il reçoive votre abonnement, il peut le copier dans son profil.

- {**Connecté**} {**Optionnels**} Message (optionnel) -> Message que vous pouvez adresser à votre contact au moment de l'envoie de son abonnement.

- Apparaître dans la liste des contributeurs -> Fait apparaître votre nom dans la liste des personnes ayant fait un don à CryptPad.

- {**Optionnels**} Ajouté un message personnalisé -> Permet d'envoyer un message pour l'équipe de CryptPad :)
