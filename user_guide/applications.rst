Nouveau document
-----------------

Pour en créer, cliquez sur `+ nouveau` dans le drive ou `Fichier > nouveau` dans un document ou `(ctrl+e)`. 


Écran de création {**Connecté**}
------------------------------------

Lors de la création d'un document, vous avez accès à son paramétrage qui vous donne plusieurs fonctionnalités :

- Être propriétaire de ce pad -> Devenez [propriétaire] de celui-ci (**conseillé**)

- Ajouter une durée de vie -> temps donné avant suppression définitive de votre document.

- Ajouter un mot de passe -> Mot de passe donné afin d'accéder au document.

- Modèle -> Choisissez un [Modèle] à utiliser.



### Fonctionnalités

#### Fonctionnalité de base

Une fois le document créé, vous avez accès à de nombreuses fonctionnalités de base, tel que :

- Les changements effectués sur votre document sont enregistrés automatiquement.

- l'affichage des utilisateurs actuellement connectés sur votre document en cliquant sur le bouton [icon] [icon].

#### Historique {**Connecté**}

Si vous avez supprimé des éléments par erreur, vous pouvez les récupérer avec l'historique de votre document.

Cliquez sur `[icon] Fichier > Historique` et sur les flèches dans la barre d'historique pour remonter dans les anciennes versions. Restaurez-les en cliquant sur [icon], sinon cliquez sur [icon] pour en sortir.

#### Modèles {**Connecté**}

Afin de ne pas recopier le même contenu d'un autre documents, vous pouvez lui appliquer un modèle qui est un gabarit réutilisable. 

Pour Transformer un document en modèle, cliquez sur `[icon] Fichier > [icon] Sauver en tant que modèle`.

Pour appliquer un modèle existant au document, cliquez sur `[icon] Fichier > [icon] Importer un modèle` (**/!\ Le contenu du document sera supprimé /!\**)
